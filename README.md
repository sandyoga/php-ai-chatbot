A simple AI chatbot built using Core PHP and OpenRouter free AI models.

## Features
- Ask questions using a single input field
- AI-generated responses
- Clean and responsive UI
- Uses free AI models (Mistral, Llama, Gemma)

## Setup Instructions

### 1. Get Free API Key
Visit: https://openrouter.ai/keys

### 2. Add API Key
Edit `config.php`:

```php
define('OPENROUTER_API_KEY', 'your_key_here');
```

### 3. Run Locally
Use XAMPP / WAMP / MAMP:

http://localhost/php-ai-chatbot-openrouter/
